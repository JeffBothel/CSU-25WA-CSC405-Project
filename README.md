# CSU-25WA-CSC405-Project
This repository is code developed for the final project in the Colorado State University Global Campus couse CSC405: Graphics and Visualization.

## References and Utilized Code
The code assembled in this repo is both from reference materials and from original code created.

All the code in the `Common/` folder came from the [course materials provided](https://interactivecomputergraphics.com/Code/). When utilized in the repo, comments will indicate where the referenced code is utilized.

**Reference:** Angel, E., & Shreiner, D. (2020). Interactive Computer Graphics (8th ed.). Pearson Education, Inc. https://www.pearson.com/store/en-us/pearsonplus/p/9780135217160.html
